Conditions

We can run, or not run code, based on if a condition is true or false. Consider this example again:

The if statement is followed by paranthesis, and inside those paranthesis is a statement that can either be true or false.

Note: Careful with semi-colons here. There is no semi-colon after the if statement or curly braces.

In our case, we compare the temperature variable (which is 0), and ask if it less than 10. In this case, 0 is less than 10, and the code inside the curley braces runs.

When we changed temperature to 20, 20 is not less than 10, so the code inside the curly braces is skipped over, and we keep running code after the braces.
