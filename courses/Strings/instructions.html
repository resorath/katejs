Strings

Variables are important to master. You can't write a useful program without them.

Variables come in many different types. So far we have been using Strings with our variables, Strings are a type of variable.

A String is a bunch of letters, numbers, and symbols that have no special meaning in JavaScript. These are all Strings, because we put them between double quotes:

  • "Kate"
  • "The brown horse went to the stable."
  • "37"
