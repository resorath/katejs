Numbers

Variables come in many different types. So far we have been using Strings with our variables.

A String is a bunch of letters, numbers, and symbols. They have no significance to JavaScript. These are all strings, because we put them between double quotes:

  • "Kate"
  • "The brown horse went to the stable."
  • "37"

However we can have other types of variables as well.
