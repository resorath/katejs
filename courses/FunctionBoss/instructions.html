End of Chapter Final Boss: Vowel Counter

Write a function called countVowels that takes a string as a parameter, and returns the number of vowels in the string. We can assume that the string is all lowercase letters, and that vowels are: a, e, i, o, u.

Hint: You can loop over a string and look at each individual character, just like an array. For example, "cat"[0] is "c". You can also use "cat".length to get the length of a string. This of course also works if the string is in a variable.
